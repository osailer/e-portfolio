About This Project

This project involved designing and implementing a backend database system for an online shopping platform. The system manages product data, orders, customers, and inventory using **SQL, Java, and PostgreSQL**.

I contributed to schema design, backend logic, and data retrieval operations, while focusing on relational modeling and SQL query optimization.
