About This Project

In CSCI Database Systems, I collaborated with Karsten Larson, Kaitlyn Nickel, and Patrick Arbach to develop a backend system for an online shopping platform. The goal was to manage products, customers, orders, and inventory while practicing relational database design with PostgreSQL and Java.

My contributions included designing the database, implementing backend logic in Java, and writing optimized SQL queries. This project taught us how to manage data relationships and ensure the system could handle ecommerce scenarios efficiently.
