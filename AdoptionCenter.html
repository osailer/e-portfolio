About This Project

This class project involved developing a full-featured adoption website for animal shelters. It includes different user roles (admin, staff, and public), a complete animal listing system, and a backend powered by **Microsoft SQL Server**.

I implemented authentication levels, user interface layouts, and database architecture to support real-world adoption workflows.
