About This Project

The Student Planner is a lightweight web application designed to help students organize assignments, track deadlines, and manage schedules. Built with **HTML, CSS, and vanilla JavaScript**, it provides dynamic task creation, editing, and completion features, all within a clean user interface.

This project strengthened my skills in DOM manipulation, event handling, and creating responsive layouts without relying on frameworks.
