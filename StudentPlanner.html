About This Project

For CSCI Modern Software Development, I collaborated with McKeon Honeyman and Jared Klevgard to create the Student Planner, a web application designed to help students organize their day-to-day lives.

Built with HTML, CSS, and vanilla JavaScript, the app allows users to add and edit tasks within their planner. It provides daily, weekly, and monthly calendar views for easy reading. We also focused on simplicity and reliability to make task management easier for students.
